Setup Guide
Connect your music sources so Claude can catalogue and manage your full library. Each connection unlocks more of the Music Command system.
Apple Music Catalogue
01
Export Apple Music library: iTunes/Music app → File → Library → Export Library (XML)
This gives you a complete list of everything in your library including purchase dates and play counts
YOU
02
Upload the XML to Google Drive in /Music/Library/apple-library.xml
YOU
03
Claudius reads the XML and imports all tracks into a Notion Music Database with title, artist, album, genre, play count
Claude handles the XML parsing — it's just a file read
CLAUDE
04
For real-time access: Apple Music API (developer.apple.com) — requires MusicKit API key
Optional. The export covers the catalogue; the API adds live playback integration
YOU
Spotify Catalogue
01
Request your Spotify data: Spotify account → Privacy Settings → Download your data
Arrives in ~5 days. Contains every stream, playlist, and saved track
YOU
02
Alternatively: Spotify API (faster). Create app at developer.spotify.com → get client ID + secret
Free, takes 10 minutes to set up
YOU
03
Clodius pulls your saved albums, playlists, and top tracks via Spotify API → stores in Notion
Claude writes the API integration code
CLAUDE
Vinyl Collection
01
Photograph your record collection — album cover facing camera, 5-10 per photo is fine
YOU
02
Send photos to Claude mobile — Claude extracts artist, title, year, genre from each cover
Works for 90%+ of albums. For obscure pressings, Claude will flag for manual entry
CLAUDE
03
Claude populates the Vinyl Collection DB in Notion and this dashboard simultaneously
CLAUDE
04
Optional: Connect Discogs account (discogs.com) for automatic valuations and edition tracking

AI Music Tools
01
Suno AI (suno.ai) — best for full song generation with lyrics. Free tier available, Pro for commercial use
Best starting point for concept albums
YOU
02
Udio (udio.com) — alternative, different voice/style model. Good for instrumental and orchestral
Try the same prompt on both — they sound very different
YOU
03

~$800-1400 for 100 copies of a standard 12" LP
