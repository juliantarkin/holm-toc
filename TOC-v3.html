System Overview
Single-operator multi-domain portfolio. The risk is context-switching cost — the Notion closeout protocol and G3 picker solve this.
The Brain
Four Claude instances covering browser, mobile, desktop, and API automation. Each has unique connector access.
→ Claudius for deep work. Clodius for automation. Claude mobile for capture. Claudio for web research.
The Memory
Notion (structured, queryable), Google Drive (documents), Obsidian (local linked lore). Each serves a distinct role.
→ Session closeouts → Notion. Long writing → Drive. Lore/world → Obsidian.
The Muscle
n8n (automation), RunPod (GPU burst), VPS (always-on), local machine (UE5/Blender), Pi 5 (TV channel).
→ VPS hosts n8n so routines run without your machine being on.
The Output
Games (Overcast Studios), Companion (Astrid), Stories (Atreal), Essays, TV Channel (KPTV-12), RPG campaigns.
→ Every project ends with a tangible artifact. If it doesn't, it's research.
Priority Tiers
Tier 1 // Unlock
n8n Foundation, Calendar Mission Setup, Claude Ecosystem Map. Do these first — they enable everything else.
Tier 2 // Build
Overcast Studios, Astrid, TIE Academy, AI GM System. Core projects that need Tier 1 infra running.
Tier 3 // Create
Essays, Atreal, Aedaeion, 5 Parsecs, SW Ships, TV Channel. High-reward creative — best after infra is stable.
Tier 4 // Sustain
Home/property tasks, finance hygiene, calendar maintenance. Monthly/seasonal. Low friction, high consequence if ignored.

The key insight: You never have to remember where you left off. Every session writes its state to Notion. Every new session reads it back. The G3 handles decision fatigue. The calendar handles scheduling pressure. Your only job is to execute the steps and log the closeout.
