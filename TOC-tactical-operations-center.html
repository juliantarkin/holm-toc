Recommended: Monarch Money vs Manual Gmail
Monarch Money (~$14.99/mo) — connects bank accounts and cards, auto-categorizes transactions, shows subscription tracker. Worth it if you want passive visibility. Best option if you want Clodius to query your spending via API.

Gmail + n8n (free) — filter receipts, auto-file to Drive. Lower overhead, requires monthly manual review. Good enough if discipline is consistent.

Recommendation: Start with Gmail + n8n (zero added cost), upgrade to Monarch if the manual process breaks down after 2–3 months.

VPS (recommended)
Always-on n8n host. ~$5–10/mo on Hetzner or DigitalOcean. Runs Astrid routines, birthday triggers, logging — 24/7 without your machine needing to be on.
Remote PC vs VPS Decision
A dedicated remote PC gives you full Windows/Linux environment, GUI apps, persistent storage — but costs $20–$100/mo and adds complexity. A VPS ($5–10/mo) running n8n covers 90% of the always-on automation needs without the overhead. For Claude Desktop specifically: Claudius on your local machine + Clodius via API handles everything a remote PC would give you.

Verdict: Start with VPS + RunPod. Revisit remote PC only if you need a persistent GUI environment you can't replicate locally.
