System Details
Option 1 — GitHub Pages (Recommended)
▸ EXPAND
01
Go to github.com → Sign in or create free account
Free forever, no credit card required
02
Create new repository → name it toc → set to Public
03
Upload TOC-v3.html and toc-hub.html → rename TOC-v3.html to index.html
index.html is what GitHub Pages serves as the root URL
04
Settings → Pages → Source: Deploy from branch → Branch: main → Folder: / (root) → Save
05
Your URL: yourusername.github.io/toc — live in ~60 seconds
06
On iPhone: open URL in Safari → Share → Add to Home Screen → name it "TOC"
This creates a full-screen web app icon on your home screen — looks and feels like a native app
Option 2 — Notion Hub Page
▸ EXPAND
01
Create a new Notion page called "TOC Hub"
02
Add a "Bookmark" block with your GitHub Pages TOC URL → this embeds a clickable link card
One tap from Notion → opens the TOC in browser
03
Add a "Database — Inline" block → name it "Tonight's Tasks" → columns: Task, Project, Claude Can Do (checkbox), Done (checkbox)
04
On iPhone: install Notion app → open the database → tap ••• → Add to home screen widget
Shows your task database as a live widget — tap to open Notion, tap the TOC link to open full command
05
Long-press iPhone home screen → + widget → Notion → select your "Tonight's Tasks" database
Medium widget (2×2) shows 4 tasks. Large (4×4) shows full database.
Option 3 — Netlify Drop (Easiest)
▸ EXPAND
01
Go to app.netlify.com/drop
02
Drag and drop your HTML files onto the page
Make sure to name your main file index.html first
03
Get a URL like random-name.netlify.app instantly — no account required
04
Create free account to claim a custom subdomain: holm-toc.netlify.app
